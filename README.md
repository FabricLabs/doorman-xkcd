# kirbi-xkcd module
A module for [Kirbi](https://github.com/richardson-media-house/kirbi), that adds commands for showing xkcd comics.

## Usage

- !xkcd [comic number]
  *example* - !xkcd => spits out latest comic
            - !xkcd 1642 => spits out comic #1642

- !highnoon
  *example* - !highnoon
